# 운영 점검표

## 환경변수 (.env.example 참조)
| 변수 | 설명 |
|---|---|
| API_PORT / API_HOST | API 서버 바인딩 |
| DATA_SOURCE_MODE | `mock`(기본) / `live` |
| SEIBRO_BASE_URL / SEIBRO_API_KEY | SEIBro 원천 |
| KOFIA_BASE_URL / KOFIA_API_KEY | 금투협 |
| KRX_BASE_URL / KRX_API_KEY | KRX/금융위 |
| DATABASE_URL / PG* | PostgreSQL |
| REDIS_URL | 캐시 |
| VITE_API_BASE_URL | 프런트→API (비우면 임베디드 Mock) |

## 배포 전 체크
- [ ] `npm install`
- [ ] `npm run test` (bond-engine + web 단위)
- [ ] `npm run build` (타입체크 + Vite 빌드)
- [ ] `npm run test:e2e` (360/390/430 Playwright)
- [ ] `.env` 실제 비밀키 주입(코드 하드코딩 금지)
- [ ] DB 마이그레이션 `db/migrations/001_init.sql` 적용
- [ ] PWA manifest·Service Worker 생성 확인(dist)

## 관측성 지표 (data_ingestion_logs)
- 수집 성공률 / 결측률 / 지연시간(latency_ms)
- 원천별 오류 카운트, 마지막 성공 시각

## 비기능 목표(PRD §7)
| 항목 | 목표 |
|---|---|
| 첫 화면 | 3초 이내 |
| 캐시 조회 | 1초 이내 |
| API P95 | 1.5초 |
| 계산 응답 | 300ms 이내 |
| 가용성 | 월 99.5% |

## 배치
- 일별 시가평가·곡선·시가표 수집: Node Cron / 서버 스케줄러
- 실행: `DATA_SOURCE_MODE=live node apps/api/dist/jobs/ingest.js` (실원천 연동 후)
- 현재는 Mock 이 실시간 생성하므로 배치 불필요
