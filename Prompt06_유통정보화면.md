특정 채권의 일별 유통정보 화면을 구현하라.

필수 조회값:
- 평가일
- 시가평가수익률
- 평가가격
- 실제 거래수익률
- 실제 거래가격
- 거래량
- 데이터 출처

차트:
- 수익률 선
- 가격 선
- 거래량 막대
- 1M/3M/6M/1Y/전체/직접지정
- 날짜 선택 툴팁

수익률 기본값 우선순위:
1. 시가평가수익률
2. 실제 거래수익률
3. 발행수익률
4. Curve 수익률
5. 사용자 입력

대체값을 사용할 때 화면에 명시하고, 결측값을 연결선으로 임의
보간하지 말라.

선택한 일자·수익률·가격을 투자 시뮬레이션으로 전달하는
‘이 수익률로 계산’ 버튼을 구현하라.
