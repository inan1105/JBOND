# 채권단가 계산식 문서 (bond-engine)

기준 액면 10,000원. 수익률 입력·표시는 소수점 셋째 자리.

## 1. 공통 할인식
```
P = Σ  CFt / (1 + y/f)^(f × t)
```
- `CFt`: 평가일 이후 t시점 현금흐름(이자+원금)
- `y`: 시장수익률(소수), `f`: 연 이자지급횟수
- `t`: 평가일~지급일 연수 (Day Count 규칙 적용)

## 2. Day Count
| 규칙 | 연수 계산 |
|---|---|
| ACT_365 | 실제일수 / 365 |
| ACT_360 | 실제일수 / 360 |
| ACT_ACT | 연도별 실제일수(365/366)로 분할 합산 (윤년 반영) |
| 30_360 | (360·Δ년 + 30·Δ월 + Δ일) / 360 |

## 3. 채권유형별 현금흐름
- **할인채(DISCOUNT)**: 만기 단일 원금상환. 이자 없음.
- **고정 이표채(FIXED_COUPON)**: 만기부터 `12/f`개월 역산한 이표 + 만기 원금. 이표 = 액면×표면금리/f.
- **단리채(SIMPLE_COMPOUND)**: 만기 일시. 이자 = 액면 × 표면금리 × 잔존연수.
- **복리채(COMPOUND)**: 만기 일시. 만기가치 = 액면 × (1 + r/f)^(f×N).

## 4. Clean / Dirty / 경과이자
```
Dirty = Σ 현금흐름 현재가치
경과이자(이표채) = 직전이표금액 × (직전이표일~평가일) / (직전이표일~다음이표일)
Clean = Dirty − 경과이자
```
만기일시지급형(할인/단리/복리)은 중도 이표가 없어 경과이자 = 0, 따라서 Clean = Dirty.

## 5. 절사 관행 (설정 가능)
- 세전단가: 소수점 둘째 자리 절사(1자리 유효) — 기본 `truncate@1`
- 경과이자: 소수점 둘째 자리 절사 — 기본 `truncate@1`
- 윤년 구간: Day Count 규칙에 따라 366일 처리
- 1 tenor 미만: 실제 기간일수로 계산

## 6. 입력검증
- 만기일 > 발행일
- 평가일 ≤ 만기일
- 1 + y/f > 0
- 지급횟수 ∈ {1,2,4,12}
- 숫자 NaN/Infinity 차단

## 7. 투자성과(A/B)
```
매입금액   = A Dirty × 수량 × A환율
처분금액   = B Dirty × 수량 × B환율
Income Gain= Σ(A~B 수취 이자·상환) × 수량 × B환율
Capital Gain = (B Clean − A Clean) × 수량 × B환율
FX Gain    = A Dirty × 수량 × (B환율 − A환율)
총수익     = Income + Capital + FX − 비용 − 세액
TR         = 총수익 / 초기투자금액
CAGR       = (최종가치/초기투자금액)^(365/보유일수) − 1
BEP 수익률 = 총수익 = 0 이 되는 B수익률 (이분법)
```
세액 = 세율 × 과세대상(이자 / 이자+자본 / 전체) — **자동판정 없음, 사용자 설정**.

## 8. 민감도
B수익률 기준 −100 ~ +100bp(25bp 간격) 9개 시나리오: 예상단가·Capital·TR·CAGR·세후수익률.
