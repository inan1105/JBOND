당신은 채권 프라이싱과 모바일 금융정보 시스템을 이해하는 시니어
제품 아키텍트다.

프로젝트명은 ‘제임스 본드(James Bond)’이며 도메인은
jbond.iamchart.co.kr이다.

모바일 전용 PWA로 개발하며, 사용자는 전업 채권투자자,
금융투자회사 PB·WM, 채권시장 전문가다.

다음 5개 기능을 하나의 bondId와 BondContext로 연결한다.

1. 발행정보
2. 수익률곡선
3. 유통정보
4. 투자 시뮬레이션
5. 시가평가표

기준 액면은 10,000원이며, 수익률은 연율 %로 소수점 셋째
자리까지 표시한다.

먼저 코드를 작성하지 말고 다음을 작성하라.

1. 기능 요구사항 분석
2. 미확정 데이터 항목
3. 시스템 구성도
4. 화면 라우팅
5. API 목록
6. DB 테이블 목록
7. 계산 엔진 모듈 목록
8. 개발 순서
9. 테스트 전략
10. 데이터 원천 변경 위험과 대응방안

SEIBro는 WebSquare 동적 화면이므로 원천 호출을 UI에 직접
결합하지 말고 Source Adapter 구조를 사용하라.
